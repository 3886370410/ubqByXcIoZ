# 智慧社区管理系统 java661

## 项目概述

智慧社区管理系统是基于SpringBoot和Vue的前后端分离系统，旨在为社区管理提供全面的信息化解决方案。

## 技术栈

### 开发环境
- IDEA
- JDK 1.8
- Maven
- Node.js

### 技术框架
- Java
- SpringBoot
- MyBatis-Plus
- Redis
- Vue

### 数据库
- MySQL

## 功能模块

### 登录模块
- 用户登录
- 用户注册

### 用户前台
- 首页轮播图
- 车位信息
- 社区信息
- 周边服务信息
- 问卷调查
- 答卷
- 留言反馈
- 个人中心
- 通知公告
- 爱心助老

### 后台管理
- 首页统计图表
- 在线报修管理
- 投诉建议管理
- 车位信息管理
- 车位租买
- 社区信息管理
- 周边服务管理
- 问卷调查管理
- 参与答卷管理
- 爱心助老管理
- 留言板管理
- 轮播图管理

## 运行环境

系统可在符合以下条件的环境中运行：
- Java 1.8 或更高版本
- MySQL 5.7 或更高版本
- Redis 服务可用

## 部署步骤

1. 准备开发环境
2. 克隆代码到本地
3. 编译后端代码，生成可执行jar包
4. 部署前端代码到Web服务器
5. 配置MySQL数据库
6. 启动后端服务
7. 访问前端页面
